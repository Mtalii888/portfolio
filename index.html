Power BI Capstone Project

An end-to-end analytics project built in Power BI — from data cleaning and modeling to an interactive dashboard for exploring key metrics.

View on GitHub →
Overview

This project involves designing and developing
an end-to-end Power BI Business Intelligence
solution for FreshChoice Mart, a supermarket
chain with multiple branches across Kenya.
The objective is to transform raw sales data into
meaningful insights through interactive dashboards and
reports that support strategic decision-making.

Approach

I extracted the data from excel,Cleaned and
transformed the data using Power Query
Built a proper Star Schema data model through
the correct relationships between tables.
Designed DAX measures for key metrics e.g
Total Sales,profit margin and total profit
Laid out the dashboard around the questions
stakeholders actually asked.

  • Data source used was Excel
  • Confirmed correct data types
  • Built DAX measures for key metrics
Key Insights
  • 1. Fresh choice westlands is the best perfoming
    store with the highest number of Total sales(419,916.47)
    and the highest sum of profits(119,119.24)
  • 2. Meat and Fish category generates the highest profit
    margin percentage (43.69%)
  • 3. All of these stores are consistently missing targets
    since the sale vs targets % are less than 1% for all of the stores
